# AEGIS Enterprise Security Platform

> AI-powered Security Architect with live threat intelligence, multi-cloud infrastructure access, persistent memory, and enterprise tool integrations.

## 🚀 Live App

**[Launch AEGIS →](https://yourusername.github.io/aegis-security)**  
*(Replace `yourusername` with your GitHub username after deploying)*

## ✨ Features

- 🌐 **Live Threat Intel** — Real-time CVE, IOC, and threat actor feeds
- 🧠 **Persistent Memory** — Remembers your environment across sessions
- ☁️ **Multi-Cloud** — AWS, Azure, GCP, and on-premise connections
- 🔗 **Integrations** — Slack, Jira, ServiceNow, Splunk, Nessus, Notion, M365
- ⚖️ **Compliance** — NIST 800-53, CIS, SOC 2, ISO 27001, HIPAA, PCI-DSS
- ⊞ **Widget Mode** — Draggable floating overlay for any workflow

## 🔑 Setup

1. Open the app URL above
2. Enter your [Anthropic API key](https://console.anthropic.com) when prompted
3. Your key is saved locally in your browser — never sent to any server

## 🔒 Security & Privacy

- API key stored in browser localStorage only
- All cloud credentials stay in your browser
- Conversations go directly to Anthropic's API
- No backend server, no data collection

## 📋 Embed in Notion

1. In any Notion page, type `/embed`
2. Paste your GitHub Pages URL
3. Click **Embed link**
